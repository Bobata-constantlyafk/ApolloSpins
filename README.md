# ApolloSpins

A graveyard-themed slot machine built with React, TypeScript, and PixiJS.

## Developer
- *Boyan Dechev*

## Stack

- **React 18** + **TypeScript**
- **PixiJS** via `@pixi/react` for all rendering
- **Vite** for bundling

## Gameplay

- Start with 100 coins and spin for 1 coin per round
- Match 2 symbols → 2× payout
- Match 3 symbols → 3× payout (Jackpot)
- Game ends when balance hits 0

## Running locally

```bash
npm install
npm run dev
```
